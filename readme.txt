Numele website: Shoe, Don`t Tell

Descriere:
Acest webste este unul de prezentare al unei firme fictve care vinde papuci(slapi)

Avem 4 pagini:

-home:
 Pagina pricipala a sitului, contine o prezentare generala a firmei.

-sevicii:
 Pagina descrie serviciile oferite de firma.

-preturi:
 Pagina afiseaza preturile pentru diferitele articole.
 Tabelul de preturi este generat dinamic folosind JavaScript si 
de asemenea si o functionalitate de calcul a pretului total.

-contact:
 Pagina continedatele de contact ale firmei, date care de alt fel pot fi accesate si din subjosul fiecarei pagini.

 Navigare:
 Site-ul dispune de un meniu prezent pe fiecare pagina care permite o navigare usoara.
